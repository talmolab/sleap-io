# Spatial annotations

!!! info "This page has been split"
    The spatial annotation types are now documented on focused pages:

    - **[Centroids](centroids.md)** — `Centroid` point annotations (object detection, TrackMate import).
    - **[Boxes](boxes.md)** — `BoundingBox` axis-aligned and rotated rectangles.
    - **[ROIs](rois.md)** — `ROI` vector polygons (Shapely, GeoJSON).
    - **[Segmentation](segmentation.md)** — `SegmentationMask` and `LabelImage` pixel-level masks and dense instance segmentation.

    See [Working with annotations in frames](index.md#working-with-annotations-in-frames)
    for how these annotations nest on [`LabeledFrame`](labels.md).
